# 🚀 Digital Marketing Landing Page

A modern and responsive landing page for a digital marketing startup, built using **HTML, CSS, and JavaScript**. This project showcases a clean UI design along with basic interactivity and form validation.

---

## 🌟 Features

✨ Hero section with call-to-action button
✨ Services section highlighting key offerings
✨ About section describing the company
✨ Client testimonials
✨ Contact form with basic validation using JavaScript
✨ Simple and clean user interface

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
